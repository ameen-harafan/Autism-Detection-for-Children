# Stimuli Directory

Please place the `geopref_1min.mp4` video file in this directory.

The video file should be:
- Named: `geopref_1min.mp4`
- Format: MP4 video
- Content: Geopref test video with social scenes on the left and geometric patterns on the right, split in the middle of the screen

Alternatively, you can place the video file directly in the `frontend/public/` directory as `geopref_1min.mp4`.
